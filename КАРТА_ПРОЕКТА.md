# 🗺️ Карта проекта LoL — что в какой папке

> **Важно:** папки с кодом названы по-английски (`data`, `scripts`, `src`, `outputs`, `notebooks`)
> и переименовывать их **нельзя** — на них ссылается весь код, ноутбуки и деплой,
> а кириллица в путях ломает Git и Streamlit. Это стандарт: код всегда на английском.
> Ниже — что каждая папка значит по-русски.

## Файлы в корне (так и должно быть)

| Файл | Что это |
|:--|:--|
| `main.py` | **Пульт управления.** Запускает весь пайплайн: `python main.py all` |
| `streamlit_app.py` | **Сам дашборд** (запуск: `streamlit run streamlit_app.py`) |
| `requirements.txt` | Список библиотек для установки |
| `.gitignore` | Что НЕ загружать в GitHub |
| `КАРТА_ПРОЕКТА.md` | Этот файл |

## Папки

```
📁 src/lol_utils/     — общий код: config.py (все настройки), metrics.py, paths.py
📁 scripts/           — скрипты пайплайна по слоям:
      riot_data_collector.py        — сбор данных из Riot API (Extract)
      build_common_analytics_layer.py — приведение к общей схеме (Transform)
      run_data_quality.py           — проверки качества данных
      build_star_schema.py          — сборка звёздной схемы
      load_to_warehouse.py          — загрузка в БД (SQLite / Supabase)
      fetch_reference.py            — справочники чемпионов и предметов
      (+ утилиты: transcribe_meeting.py — расшифровка встреч)

📁 data/              — ДАННЫЕ:
      raw/            — исходный Kaggle-файл (league_data.xlsx)
      api/            — собранное своим сбором через Riot API
      reference/      — справочники Data Dragon (champions.csv, items.csv)
      normalized/     — общая таблица всех матчей (Parquet — основной формат)
      api_first_sample/ — старый первый сэмпл (можно убрать в архив)

📁 outputs/           — РЕЗУЛЬТАТЫ:
      sql/star/       — ⭐ звёздная схема в Parquet — ЭТО ДАННЫЕ ДАШБОРДА
      sql/            — SQL-витрины (по ролям, чемпионам, предметам)
      data_quality/   — отчёт проверок качества
      api_eda/, common_eda/ — графики и агрегаты разведки данных
      warehouse/      — локальная база lol.db (SQLite)

📁 notebooks/         — НОУТБУКИ (Jupyter):
      active/         — рабочие (анализ Kaggle, API, SQL-слой, общий EDA)
      reference/      — черновики-референсы

📁 docs/              — ДОКУМЕНТЫ:
      transcripts/    — расшифровки встреч с наставником
      notes/, task/   — заметки и описание задания

📁 datalens/          — выгрузки и описание для дашборда в Yandex DataLens
📁 media/             — МЕДИА: video/ (записи встреч), audio/
📁 archive/           — архив старых версий (archive.zip)
📁 .claude/           — служебный конфиг инструментов (не трогать)
```
